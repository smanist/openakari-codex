# Decisions (ADR convention)

Decision records live in `decisions/`.

## When to write a decision record

Write an ADR when you make a choice that:

- changes conventions or schemas,
- affects how future sessions operate,
- or resolves an ambiguity that would otherwise be re-litigated.

## Minimal structure

- Title: `# 00xx: <Decision title>`
- `Date: YYYY-MM-DD`
- `Status: proposed | accepted | deprecated`
- Sections: `## Context`, `## Decision`, `## Consequences`

See also: `docs/schemas/decision-record.md`.
