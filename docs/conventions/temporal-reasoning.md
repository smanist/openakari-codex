# Temporal reasoning

This repo is read across time. Prefer absolute dates over relative time language.

## Rules

- When referencing “today / yesterday / last week”, include a concrete date (e.g., `2026-03-25`).
- For stale blockers, include the date of the blocking event in the tag (e.g., `[blocked-by: external: waiting on X (2026-03-01)]`).
- When writing logs, use the project-local date heading convention: `### YYYY-MM-DD (Title)`.
